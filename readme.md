# Gneiss

Gneiss is an open-source social network kit that allows you to build your own private social network.

---
![Gneiss Social Network Kit](assets/gneiss.png)


## Features

* **Posts**:  Create, update, delete posts
* **Likes**: Like/unlike posts

### TODO
